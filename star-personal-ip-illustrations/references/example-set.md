# Stable Example Set

These images are the canonical baseline. Use them to calibrate identity, line weight, proportions, outfit, palette, and empty space. Never copy an exact pose or composition unless the user asks for it.

- `assets/star-main-ip-reference.png`: primary `main-ip` source of truth. It fixes the regular full-body proportions, red bow on the character's left side, pink smartwatch on her left wrist, pendant, summer outfit, and small shoes.
- `assets/star-chibi-ip-reference.png`: primary `chibi-sticker` source of truth, extracted from the approved nine-grid sticker drawings. Use its huge round head, tiny short body, cobalt loose full-length pants that reach the shoes, simplified limbs, and naïve crayon treatment for page-corner accents and expressive sticker scenes.
- `assets/examples/star/03-crayon-blue-white.png`: primary `crayon-blue` reference. Use its cobalt wax-crayon texture, pure empty white background, pale-pink cheek blush, and small woven-mesh sneakers while preserving the shared summer-IP identity.
- `assets/examples/star/04-naive-crayon-character.png`: secondary naive-crayon reference for the rounder silhouette, friendly expression, handmade grain, and simplified geometric treatment. Its dark blue palette is intentionally too monochromatic to serve as the color baseline; borrow the drawing feeling, not the limited color distribution.
- `assets/examples/star/05-ai-business-improvement-scroll.png`: work and performance-review example for a left-to-right business-improvement narrative: planning and documentation → AI-assisted refinement → standardized modules → data evidence. Reuse the parchment-scroll composition only when a journey needs several stages. The red bow and pink smartwatch are permanent identity elements; the eHello emblem is a company-scene detail.
- `assets/examples/star/06-lively-yellow-portrait.png`: source reference for the permanent red bow, pink smartwatch, stronger blush, and warm high-energy palette. Use it for expressive personal portraits when a colored background is requested. The yellow background, cropped framing, and enlarged portrait proportions are not the default main-IP composition.

For other topics, keep the same character but choose a grammar from `references/personal-ip.md` that fits the current idea.
