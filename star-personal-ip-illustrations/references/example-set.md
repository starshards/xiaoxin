# Stable Example Set

These images are the canonical baseline. Use them to calibrate identity, line weight, proportions, outfit, palette, and empty space. Never copy an exact pose or composition unless the user asks for it.

- `assets/star-ip-character-sheet.png`: primary source of truth for the face, hair, body proportions, front/side/back views, common actions, and expressions.
- `assets/star-summer-ip-reference.png`: primary standalone summer-IP reference for everyday image generation; it preserves the approved card-holding pose and uses the corrected smaller shoes.
- `assets/star-summer-ip-single.png`: stable named copy of the approved standalone character.
- `assets/star-work-ip-reference.png`: work-context reference with the red conch company emblem placed only on the held card. Use it for company, project, delivery, or performance-review scenes; do not add the emblem to unrelated personal illustrations.
- `assets/examples/star/01-card-gold-line-spark.png`: reference only when the content is specifically about connecting scattered pieces into a working or verified result; it intentionally uses the same approved standalone pose.

For other topics, keep the same character but choose another grammar from `references/personal-ip.md`. The card, gold line, and sparkle are not mandatory identity features.
